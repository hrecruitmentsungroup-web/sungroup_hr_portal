# SunGroup Recruitment

ระบบรับสมัครงานออนไลน์ของ SunGroup — ผู้สมัครกรอกใบสมัครผ่านเว็บ ฝ่าย HR จัดการและคัดเลือกผ่านระบบหลังบ้าน

🔗 **เว็บไซต์:** https://hrecruitmentsungroup-web.github.io/sungroup_hr_portal/

---

## ระบบทำงานยังไง

```
ผู้สมัคร ──กรอกใบสมัคร──▶ Supabase (ฐานข้อมูล) ──▶ HR Portal ──จัดการ──▶ อัปเดตสถานะ
   (index.html)              + Storage (รูป)         (sungroup-hr-portal.html)
```

- ผู้สมัครกรอกฟอร์ม + แนบรูป → ข้อมูลถูกบันทึกลง Supabase ทันที
- HR เปิด Portal เห็นผู้สมัครทั้งหมดแบบเรียลไทม์ คัดเลือกได้ทีละขั้น
- ทุกการเปลี่ยนแปลงบันทึกกลับ Supabase และ sync ทุกหน้า

## 2 ส่วนหลัก

| ไฟล์ | สำหรับใคร | ทำอะไร |
|------|-----------|--------|
| `index.html` | ผู้สมัครทั่วไป | ดูตำแหน่งงาน + กรอกใบสมัคร (5 ขั้นตอน ตามฟอร์ม FR-HR-01-032) + แนบรูป |
| `sungroup-hr-portal.html` | ฝ่าย HR | จัดการผู้สมัคร, ดูใบสมัครฉบับเต็ม + พิมพ์ PDF, วิเคราะห์ทักษะ, จัดการตำแหน่ง, export |

### ฟอร์มใบสมัคร (อิงใบสมัครกระดาษ FR-HR-01-032 REV.10)

ผู้สมัครกรอก 5 ขั้นตอน:

1. **ข้อมูลส่วนตัว** — ตำแหน่งที่สมัคร 2 อันดับ, สถานที่ปฏิบัติงาน, วันเริ่มงาน, เงินเดือน + ข้อมูลส่วนตัวเต็ม (เลขบัตรประชาชน, ที่อยู่, สถานภาพ, ทหาร, ผู้ติดต่อฉุกเฉิน ฯลฯ)
2. **ครอบครัว/การศึกษา** — ตารางครอบครัว (บิดา มารดา พี่น้อง คู่สมรส) + ตารางการศึกษาแยกระดับ
3. **ประสบการณ์** — ประวัติการทำงาน, ฝึกงาน/อบรม, Project
4. **ทักษะ** — คะแนนทักษะ 6 ด้าน, ภาษา (อ่าน/เขียน/พูด), คอมพิวเตอร์, ยานพาหนะ, บุคคลอ้างอิงในเครือ, สุขภาพ, เหตุผล/ความคาดหวัง
5. **ตรวจสอบ** — สรุปข้อมูล + คำรับรองผู้สมัคร (บังคับติ๊กก่อนส่ง)

การเก็บข้อมูล: ช่องหลักลงคอลัมน์เดิมของตาราง `applicants` ส่วนช่องเพิ่มเติมทั้งหมดเก็บเป็น JSON ในคอลัมน์ `form_data` (jsonb)

ฝั่ง HR Portal: **คลิกชื่อผู้สมัคร**ในหน้ารายชื่อ → เปิดป๊อปอัพใบสมัครฉบับเต็ม พร้อมปุ่ม **🖨️ พิมพ์ / บันทึก PDF** (ใช้หน้าพิมพ์ของเบราว์เซอร์ เลือก Save as PDF ได้)

## การใช้งานเบื้องต้น (HR)

1. **เข้าระบบ** — เปิด HR Portal แล้วล็อกอิน
2. **Dashboard** — ดูภาพรวมจำนวนผู้สมัครและสถิติ
3. **รายชื่อผู้สมัคร** — ดูตาราง กรองตามสถานะ ค้นหา (แสดงทีละ 10 หน้า เรียงคนสมัครล่าสุดก่อน)
4. **คัดเลือก** — กดปุ่มเลื่อนสถานะตามลำดับ:
   ```
   รอพิจารณา → สัมภาษณ์รอบ 1 → นัดสัมภาษณ์ → ผ่านคัดเลือก
                    (ทุกขั้นกด "ไม่ผ่าน" ได้)
   ```
5. **วิเคราะห์ทักษะ** — ดูกราฟเรดาร์คะแนน 6 ด้านของแต่ละคน เปรียบเทียบและคัดเลือกได้
6. **Export Excel** — ดาวน์โหลดข้อมูลผู้สมัครทั้งหมด (พร้อมลิงก์รูป)
