# SunGroup Recruitment

ระบบรับสมัครงานออนไลน์ของ SunGroup — ผู้สมัครกรอกใบสมัครผ่านเว็บ ฝ่าย HR จัดการและคัดเลือกผ่านระบบหลังบ้าน

🔗 **เว็บไซต์:** https://hrecruitmentsungroup-web.github.io/sungroup_hr_portal/

---

## ระบบทำงานยังไง

```
ผู้สมัคร ──กรอกใบสมัคร──▶ Supabase (ฐานข้อมูล) ──▶ HR Portal ──จัดการ──▶ อัปเดตสถานะ
   (index.html)              + Storage (รูป)         (sungroup-hr-portal.html)
```

- ผู้สมัครกรอกฟอร์ม + แนบรูป → ข้อมูลถูกบันทึกลง Supabase ทันที
- HR เปิด Portal เห็นผู้สมัครทั้งหมดแบบเรียลไทม์ คัดเลือกได้ทีละขั้น
- ทุกการเปลี่ยนแปลงบันทึกกลับ Supabase และ sync ทุกหน้า

---

## 2 ส่วนหลัก

| ไฟล์ | สำหรับใคร | ทำอะไร |
|------|-----------|--------|
| `index.html` | ผู้สมัครทั่วไป | ดูตำแหน่งงาน + กรอกใบสมัคร (5 ขั้นตอน) + แนบรูป |
| `sungroup-hr-portal.html` | ฝ่าย HR | จัดการผู้สมัคร, วิเคราะห์ทักษะ, จัดการตำแหน่ง, export |

---

## การใช้งานเบื้องต้น (HR)

1. **เข้าระบบ** — เปิด HR Portal แล้วล็อกอิน
2. **Dashboard** — ดูภาพรวมจำนวนผู้สมัครและสถิติ
3. **รายชื่อผู้สมัคร** — ดูตาราง กรองตามสถานะ ค้นหา (แสดงทีละ 10 หน้า เรียงคนสมัครล่าสุดก่อน)
4. **คัดเลือก** — กดปุ่มเลื่อนสถานะตามลำดับ:
   ```
   รอพิจารณา → สัมภาษณ์รอบ 1 → นัดสัมภาษณ์ → ผ่านคัดเลือก
                    (ทุกขั้นกด "ไม่ผ่าน" ได้)
   ```
5. **วิเคราะห์ทักษะ** — ดูกราฟเรดาร์คะแนน 6 ด้านของแต่ละคน เปรียบเทียบและคัดเลือกได้
6. **Export Excel** — ดาวน์โหลดข้อมูลผู้สมัครทั้งหมด (พร้อมลิงก์รูป)

## เทคโนโลยี

- **Frontend:** HTML + CSS + JavaScript (vanilla, ไฟล์เดียวจบ)
- **Database & Storage:** Supabase
- **Hosting:** GitHub Pages (push ขึ้น `main` = deploy อัตโนมัติ)
